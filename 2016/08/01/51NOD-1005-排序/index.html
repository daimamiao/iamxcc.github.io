51NOD 1005: 排序

51NOD

题目描述：给出N个整数，对着N个整数进行排序。

input

第1行：整数的数量N（1 <= N <= 50000)
第2 - N + 1行：待排序的整数（-10^9 <= A[i] <= 10^9）

output

共n行，按照递增序输出排序好的数据。

input示例

5
5
4
3
2
1

output示例

1
2
3
4
5
