WinSnap:带阴影的截图才是好截图

Win10
热度 ℃ 18天前
Win105 工具1

我认为 OS X 内建的截图工具的最大优点就是自动加阴影。正所谓「再小的需求也应该被满足，是的，你可能发现了上文中所有的截图都是带有阴影的，真的就是 OS X 截图的那种阴影诶！

如果你也像我一样对这个阴影喜爱有加，你一定需要 WinSnap 这款应用。仅仅 2MB，你便能够在 Windows 上完全实现 OS X 内建截图的效果，阴影和快捷键都不在话下，有过之无不及。

下载地址是：WinSnap ,这个是免安装版本的，下载后可以直接用了。
